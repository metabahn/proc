Usage: proc [global options] <subcommand> [args]

Available commands:
  compile       Compiles a supported source file to a Proc AST.
  run           Runs a supported source file in Proc.
  version       Show the current version of this command-line interface.

Global options:
  -auth=AUTH    The authorization to use when interacting with Proc.
  -help         Show this help output, or the help for the given command.
